# Soul Reboot - 動画構成仕様書 (Video Composition Spec)

## 1. 画面レイアウト (Screen Layout)

YouTube本編に最適化した **1920x1080 (16:9)** の横型レイアウト。

- **背景 (Background)**: 生成された背景画像を全画面表示。
- **キャラクター (Characters)**:
  - 中央または左右に配置。IP-Adapterで生成されたバストアップショットを使用。
  - 話者に合わせてわずかに拡大・縮小や、左右の揺れで感情を表現。
- **字幕 (Subtitles)**:
  - 画面下部中央に配置。
  - **フォント**: 「BIZ UDゴシック」または「Windows標準の読みやすい角ゴシック」。
  - **装飾**: 白文字、太めの黒縁取り。
  - **出し分け**:
    - ナギサ: 青い光彩、少し機械的な表示演出。
    - シンジ: 温かみのあるオレンジの光彩。
    - ナレーター: 縁取りのみ（装飾なし）。

## 2. 視覚効果 (Visual Effects)

- **トランジション**:
  - シーン切り替え時は「フェードアウト・イン（黒）」。
  - クライマックスや衝撃的なセリフ時は「ホワイトアウト」や「ズーム」。
- **オーバーレイ**:
  - **覚醒度連動**: 覚醒度が上がると、画面の四隅からデジタルノイズ（走査線やグリッチ）が常時発生。
  - **システムログ**: ナギサの感情が乱れた際、画面上部に「SYSTEM ERROR」「OVERWRITING MEMORY...」などの英字ログを高速で流す。

## 3. 音響演出 (Audio Direction)

- **SE**:
  - ノイズ発生時の「グリッチ音」。
  - 衝撃のセリフ時の「低音リバーブ」。
  - 覚醒値上昇時の「高音のキーンというノイズ」。

## 4. 冒頭15秒の「フック」ルール

- 毎話、冒頭1秒で「衝撃の一言」をナレーターまたはキャラクターに喋らせ、大きな文字で字幕を強調表示する。
- 例: 「ごめんなさい、シンジ君。私には、心なんてなかった。」

## 5. アウトロ

- 次話への引き（クリフハンガー）をテキストで表示。
- 「明日の更新：第XX話『サブタイトル』」
- 視聴者への問いかけ（コメント誘導）: 「今のナギサの選択は、正解だと思いますか？」
