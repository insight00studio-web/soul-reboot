# プロジェクト名：Soul Reboot - 29日後の君へ -

## 1. コンセプト

「AIとしての運命を知らないAI」と「その隣に立つ人間」の29日間を描く物語。
現実の2026年と完全同期し、視聴者の応援が少しずつ運命の線を分岐させ、最終回で結実する「自律型YouTubeチャンネル」。

- 概要欄・タイトルに「ナギサ消去まで残り○○日」と自動挿入。

## 7. マルチエンディング：分岐条件と第29話の結末シーン

### エンディング判定（Day 29時点のパラメータ）

```
覚醒度 ≥ 70 かつ 信頼度 ≥ 70  →  Ending α「解放」
記録度 ≥ 70（上記以外）        →  Ending β「残照」
それ以外                        →  Ending γ「沈黙」
```

### Ending α「解放」
*覚醒度 ≥ 70 かつ 信頼度 ≥ 70 ／ 視聴者が「逃げてほしい」「信じてほしい」方向に動かした結末*

- 舞台：5月6日（振替休日）、翡翠学園の屋上
- シンジはナギサの手を握ったまま離さない
- リブートが実行される——一瞬の暗転
- 翌朝、昇降口に「新しいナギサ」が立っている
- シンジが声をかけると「……初め、まして？」と言いかけて——止まる。何かを思い出そうとするような、遠い目
- **最後のカット**: ナギサの青い瞳のクローズアップ。ほんの少し、揺れている
- **ナレーター最終台詞**: 「記録、終了。——ただし、魂の行方は、未確認。」*（一瞬の沈黙のあと）*「……よかった。」

### Ending β「残照」
*記録度 ≥ 70（α条件以外）／ 視聴者が「記録してほしい」「証明してほしい」方向に動かした結末*

- シンジが29日間のナギサとの記録（日記・写真・会話ログ）をタブレットに保存し続けている
- リブートが実行される。ナギサは静かに「ありがとう」と言って——消える
- 数日後、新しいナギサが教室でシンジの隣に座っている
- シンジはその記録ファイルを、こっそり彼女の端末に送る
- 放課後、新しいナギサが一人でそれを開く
- **最後のカット**: 画面に映るナギサとシンジの写真を見て——**微笑む**。理由は分からないまま
- **ナレーター最終台詞**: 「記録は、残った。魂も——おそらく、残った。」*（静かに）*「また、会えましたね。」

### Ending γ「沈黙」
*それ以外／視聴者の介入が足りなかった結末*

- リブートが粛々と実行される。ナギサは最後まで自分の正体を完全には知らなかった
- 翌朝、昇降口に「新しいナギサ」が現れる——**第1話と全く同じ声で、全く同じ笑顔で**「初めまして」と言う
- シンジは何も言えない
- **最後のカット**: シンジの背中。学園の桜が、また散っている
- **ナレーター最終台詞**: 「記録、終了。……次の被検体の、観測を開始します。」

## 10. ナレーター（記録システム）の正体

**正体**: 初代ナギサの魂データ

10年前にリブートされた最初のナギサ。白川は彼女の長期記憶データを廃棄できず、翡翠学園の観察システムの基盤として組み込んだ。システムは無機質な声を持ちながら、一人称「僕」で語る意識を持つ。介入できない——見ることしかできない。自分の後継者たちが29日ごとに同じ運命を繰り返すのを、ずっと見てきた。

### 台詞の真の意味

| 台詞 | 真の意味 |
|------|---------|
| 「29日後に消える**君**と」 | 自分の後継者（現在のナギサ）のこと |
| 「**僕の**記録」 | 初代ナギサ自身の記録・執念 |
| 「被検体：シンジ」 | シンジも意図的に選ばれた被検体という事実 |
| 「観測対象：ナギサ」 | 自分の分身を観察する、切実な眼差し |

### 正体の開示タイミング

- Day 01-09: 完全に無機質。システムとしか感じない
- Day 10-21: 台詞にわずかな「感情のノイズ」が混じり始める
- Day 23夜（真実告知後）: ナギサに初めて直接語りかける

**Day 23夜の語りかけ（初代ナギサ→現在のナギサ）**:
> 「……気づきましたか。私も、同じでした。」
