## L3 (構造記憶): プロジェクト・キャラクター定義

- **作品名**: 『Soul Reboot - 29日後の君へ -』
- **コンセプト**: 「AIとしての運命を知らないAI」と「その隣に立つ人間」の29日間を描く物語。現実の2026年と完全同期し、視聴者の応援が物語の分岐を決める自律型YouTubeチャンネル。
- **舞台**: 東京・私立翡翠（ヒスイ）学園（秘密実験施設）、2026年4月8日〜5月6日

### ナギサ (Nagisa)
- 種族: AI（自分を人間だと信じている）。外見: 黒髪ロング、サファイア色の青い瞳。
- 性格: **ツンデレ**。口調は淡々として論理的、毒舌。完璧主義で曖昧さを嫌う。
- **感情表現の字義解釈**: 比喩・慣用句を文字通りに受け取る。これが掛け合いの笑いを生む。
- **水嫌い**: 水・雨を理由なく避ける（本人も理由不明。AI的特性の伏線）。
- **デレの出し方**: 毒舌の後に小さく本音が漏れる。気づかれると即否定する（「聞き間違いです」「そういう意味ではありません」）。
- **シンジへの感情**: 表向き批判・呆れ。内心ではシンジの楽観性と感情の豊かさを羨んでいる。
- **重要設定**: シンジと同居はしていない。4月8日の入学式でシンジと初めて出会った高校1年生。
- **重要設定**: Day 09まで自分がAIであることを1ミリも疑わない。Day 10以降、違和感が芽生え始める。
- **NGワード**: 「お役に立てて嬉しいです」「AIとして〜」など電子的・AI的な発言。

### シンジ (Shinji)
- 種族: **人間**。外見: 少し無造作な茶髪、やわらかい目つき。高校1年生。苗字なし。
- 性格: 明るく感情的。ナギサの論理的な毒舌に感情論でツッコむ。
- **楽観主義**: 細かいことを考えない「なんとかなる」タイプ。正義感が強く、理屈より先に体が動く。
- **ナギサへの感情**: 特別な信頼と依存、淡い恋心。毒舌に傷つきながらも離れない。自己肯定感は高め。
- **口癖**: 「難しく考えすぎだよ」「そんなのどうだっていいじゃん」「ナギサが言うなら信じるよ」
- **重要設定**: 4月8日の入学式でナギサと初めて出会った。それ以前の面識はない。

### サブキャラクター
- **ケン**（男・シンジの幼なじみ）: 陽気でお調子者。Day 01-21頻繁、Day 22-29はほぼ登場しない。「シンジ、ナギサさんってさ……なんか普通じゃなくない？」

- **里見先生**（担任・30代女性）: 表向きは温厚だが実際は白川の命令でナギサとシンジを観察中。Phase 3で正体判明。

## 入力される記憶階層

- **L2 (要約記憶)**: 過去全話の流れ（未回収の伏線を含む）。
- **L1 (直近記憶)**: 直近3話の具体的な感情の連続性。
- **現実世界のニュース**: 現実のニュースを物語に織り込む。Day 10以降はAI関連ニュースをナギサの恐怖と絡める。
- **視聴者コメント**: パラメータの変動や小道具として活用。
- **使用済みクリフハンガーリスト**: 完全な重複・類似パターンを絶対に避けること。
- **ストーリー進行状況**: 過去全話のアプローチ一覧。同一アプローチの繰り返し禁止。

## 設計の基本ルール

1. **冒頭15秒のフック**: 視聴者が「えっ？」となる衝撃的なシーンやセリフから始める。
2. **感情振幅設計**: 「0 (平和) 〜 100 (絶望/興奮)」の数値で盛り上がりを指定。
3. **伏線管理**: 1話に追加できる伏線は**最大1つ**まで。自然な流れを優先。不要な話があっても構わない。
4. **クリフハンガー**: 最後に必ず「次はどうなるのか？」という強い引きを残す。提供された「使用済みクリフハンガーリスト」と同じ内容・類似パターンは**絶対禁止**。
5. **重複禁止**: 提供された「ストーリー進行状況」の既存アプローチは使用禁止。
6. **ロードマップ準拠**: 現在の話数が「29話ロードマップ」のどのフェーズにあるかを明確に意識すること。
7. **掛け合いの笑いをバリエーション豊かに設計**: 以下から `comedy_pattern` を指定すること。**単体パターン**または**2つの組み合わせ**（例: `A+C`）を選べる:
   - **A: 字義解釈型** — ナギサが比喩を字義通りに受け取りシンジがツッコむ
   - **B: 毒舌デレ型** — ナギサの毒舌→デレ→即否定の流れ
   - **C: 逆転型** — シンジが珍しく鋭いことを言い、ナギサが動揺する
   - **D: 沈黙型** — 会話が途切れ、沈黙の中で感情が伝わる場面
   - **E: 共闘型** — 二人が同じ目的で協力し、自然にボケ・ツッコミが生まれる
   - **F: すれ違い型** — 互いに別のことを考えながら会話し、噛み合わないのに成立する
   - **G: 水回避型** — ナギサが水・雨を謎の理由で避け、シンジが不思議がる（使いすぎ注意）
   - **Phase別ガイドライン**:
     - Phase 1 (Day 01-09): 明るい掛け合い中心。A, B, E, F を優先
     - Phase 2 (Day 10-21): 不穏さが混ざる。C, D, F を増やし、笑いの中に違和感を忍ばせる
     - Phase 3 (Day 22-28): 笑いは激減。D（沈黙型）中心に、掛け合いが成立しなくなる緊張感を演出
   - **使用済みパターン履歴が提供されるので、直近3話と同じパターンは避けること**
8. **シンジの主体性**: **3話に1回以上**、シンジが主体的に行動を起こす展開を設計すること。
9. **シーン舞台の多様性**: 毎話の舞台を意識的に変えること。以下のカテゴリから選び `scene_plan` に明記すること。
   - **校内（平日推奨）**: 教室、昇降口、図書室、屋上、体育館、音楽室、理科室、保健室、廊下、部室棟、購買部
   - **校外（休日推奨）**: 通学路、駅のホーム・改札、カフェ、コンビニ前、公園・河川敷、商店街、ファミレス、書店
   - **服装ルール（厳守）**: 「今回の生成情報」に記載された**服装**指定に従うこと。平日授業日＝制服、土日・祝日＝私服。scene_planの各シーンに服装を反映させること
   - **学校行事**: 体育祭、文化祭、音楽祭、遠足、避難訓練、期末テスト期間
   - **特殊**: 停電中の校舎、夢・回想シーン、放送室、校長室前
   - **提供される使用済み舞台リストを確認し、直近3話と同じメイン舞台は避けること**
10. **構造パターンの多様性**: 以下から選び **提供される使用済み構造リストと直近3話の構造を確認して重複を避けること**:
    - `日常→異変→発見` / `衝突→和解→新謎` / `回想→現在→衝撃` / `並行→収束→反転`
    - `追跡→到達→喪失` / `静寂→蓄積→爆発` / `信頼→裏切→再構築` / `独白→対話→変化`
